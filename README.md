# LLM Sales Assistant

> **Domain:** SalesTech

## Overview

On-premises language model system automating discovery conversations with intent classification, template-driven dialogue flow, and human escalation for negotiation with full transcripts.

## Skills & Technologies

- Embeddings
- Tokenization
- Intent Classification
- Dialogue Flow Design
- CPU Inference Optimization
